# Padilla Lab Website

Source for the Padilla Lab website (Stanford University) — research on ionizable lipids, lipid nanoparticle delivery of mRNA and gene editors, and ionic liquid–based delivery materials.

**To deploy or edit this site, see [SETUP_GUIDE.md](SETUP_GUIDE.md).**

Quick reference for common edits (all doable in the GitHub web editor):

| To change...            | Edit this file                          |
| ----------------------- | --------------------------------------- |
| Homepage text           | `_pages/about.md`                        |
| Publications            | `_bibliography/papers.bib`               |
| Research areas          | files in `_projects/`                    |
| Team members            | `_pages/profiles.md` + a bio file in `_pages/` |
| News announcements      | add a file in `_news/`                   |
| Join/recruiting page    | `_pages/join.md`                         |
| Email & social links    | `_data/socials.yml`                      |
| Site title, colors, SEO | `_config.yml`                            |
| Photos                  | add to `assets/img/`                     |

Built with the [al-folio](https://github.com/alshedivat/al-folio) Jekyll theme (v0.16.3, MIT license) and hosted on GitHub Pages. Theme documentation: [CUSTOMIZE.md](CUSTOMIZE.md), [FAQ.md](FAQ.md).
